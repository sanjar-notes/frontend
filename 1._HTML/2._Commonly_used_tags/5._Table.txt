====== 5. Table ======
Created Tuesday 01 September 2020

=== Tags ===
1. **''table''**- basic table tag
2. **''tr''** - row of table.
3. Cells
	a. **''td''** - tag for enclosing a usual data cell.
	b. **''th''** - tag for enclosing a header cell, i.e this is only used inside **''thead''**
4. Sections - //optional//.
	a. **''thead'' **- head of the table, containing headings
	b. **''tbody''** - body of the table, containing cells
	c. **''tfoot''** - the foot of the table, contains appendices and/or legends.

* **tfoot** may be ommitted.
{{{code: lang="html" linenumbers="True"
<table>
	<thead>
		<tr>
			<th>Name</th>
			<th>RollNo</th>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td></td>
		<tr>
	</tbody>
</table>
}}}
