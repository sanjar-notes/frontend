====== 3. flex-basis ======
Created Thursday 03 September 2020

Sets the intitial size of an item which is irrespective of the flex-grow and flex-shrink property
{{{code: lang="css" linenumbers="True"
p
{
	flex-basis:200px;
}
}}}
