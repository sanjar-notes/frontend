====== 2. Writing stuff ======
Created Wednesday 17 June 2020

=== Basic text constructs ===
1. <h1> ⇒ heading, h1 to h6 are available, with h6 being the smallest(least important).
2. <p> ⇒ paragraph tag
3. <cite> ⇒ citation (italicized)
4. <pre> ⇒ pre-formatted or verbatim text
5. <blockquote cite> ⇒ indented
6. <q> ⇒ Italicized
7. <abbr></abrr> ⇒ abbreviation. Short is new, <acronym> is obsolete.
8. <dfn></dfn> ⇒ definition.

=== Code and scientific notation ===
1. <code><code/> tag. Mostly the code is wrapped by a <pre> wrapped by a <code> element.
2. <samp><samp/> tag. The output of the code.
3. <var><var/> represents a variable.
4. <sub></sub> is used for subscript text.
5. <sup></sup> is used for superscript text.

=== Miscellaneous ===
11. <kbd> Ctrl+S<kbd> for keybindings.
12. <address></address> ⇒ italicizes the address
13. <br /> ⇒ newline
14. <hr /> ⇒ visible horizontal separator
--------------------
Note: 
* Use the **title** attribute for abbreviations and citations.
* Except ''hr'' and ''br'', all other tags have a text content, so they are closed tags.
Demo
{{{code: lang="html" linenumbers="True"

=== Handling Whitespace ===
* As discussed before, browser ignores whitespace in HTML.
* Whitespace, operators and emojis are handled using ''entitites''.
* Syntax for entities - there are 3 kinds of notations - word(&entity_name), decimal and hex(&#entity_number).
* How to use - just write the entity in the content.
